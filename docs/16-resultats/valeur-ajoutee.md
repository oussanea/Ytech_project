# 💎 Valeur Ajoutée

> Récapitulatif des services déployés, des compétences développées et de la valeur pédagogique du projet Ytech Solutions.

---

## Infrastructure déployée — Récapitulatif

| Service | Technologie | Valeur métier |
|---|---|---|
| **YtechBot** | Ollama + llama3.2 + Streamlit | Assistant IA interne, 100% local, sans fuite de données |
| **App CRUD RH** | PHP + MariaDB | Gestion numérique des 24 employés Ytech |
| **App Web** | Laravel + ModSecurity | Vitrine sécurisée avec protection WAF |
| **Zabbix** | Monitoring agents | Supervision en temps réel de toute l'infra |
| **Wazuh SIEM** | Elasticsearch | Corrélation d'événements + détection d'intrusion |
| **Grafana SOC** | Dashboards temps réel | Centralisation de tous les indicateurs de sécurité |
| **Nessus** | Scanner de vulnérabilités | Identification proactive des failles |
| **Bitwarden** | Gestionnaire de mots de passe | Élimination des mots de passe faibles/partagés |
| **Headscale** | Zero Trust VPN | Accès distant sécurisé sans VPN traditionnel |
| **OPNSense** | Firewall + Suricata IDS | Première ligne de défense + détection d'intrusion |
| **Backup AES-256** | rsync + rclone + Google Drive | Continuité d'activité garantie |

## Valeur pédagogique

Ce projet constitue une infrastructure complète de niveau **professionnel** couvrant :

- 🏗️ Architecture en couches (DMZ → App → DB → MGMT → Users)
- 🔐 Sécurité défensive (prévention + détection + réponse)
- 📊 Observabilité totale (métriques + logs + alertes)
- 🤖 IA locale sécurisée (Ollama sans dépendance cloud)
- 🔄 DevOps (Git, Docker Compose, scripts automatisés)
- 📋 Conformité (ISO 27001, règle 3-2-1, Zero Trust)
