# Projet Final JobInTech : Infrastructure et Sécurité Ytech Solutions 

## 1. Contexte du Projet
Ce projet consiste en la refonte complète de l'infrastructure informatique de **Ytech Solutions**, entreprise spécialisée dans les services numériques et le développement web.

## 2. Objectifs du Projet
* Héberger l'application web commerciale de vente de packs.
* Déployer une application CRUD dédiée au département Ressources Humaines.
* Sécuriser les services et les machines (Hardening).
* Simuler l'infrastructure réseau via GNS3 ou Packet Tracer.
* Intégrer des pratiques DevOps (CI, versioning, GitHub).
* [Identifier et corriger les faiblesses de sécurité via des tests d'intrusion.

## 3. Architecture Technique
* **Système d'exploitation :** Linux (Ubuntu Server).
* **Serveur Web :** Nginx.
* **Langages et Frameworks :** PHP (Laravel / Bagisto).
* **Base de données :** MySQL.

## 4. Mesures de Sécurité
* Alignement sur les bonnes pratiques de la norme **ISO 27001**.
* Chiffrement des données en transit via **HTTPS / TLS**.
* Segmentation réseau par **VLAN** et mise en place d'une **DMZ**.
* Application du principe du moindre privilège et du **Zero Trust**.

## 5. Documentation technique
La documentation complète du projet, incluant les schémas d'infrastructure et les rapports de vulnérabilités, est réalisée avec **Docusaurus**.
