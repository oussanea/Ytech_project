# [cite_start]Projet Final JobInTech : Infrastructure et Sécurité Ytech Solutions [cite: 1]

## 1. Contexte du Projet
[cite_start]Ce projet consiste en la refonte complète de l'infrastructure informatique de **Ytech Solutions**, entreprise spécialisée dans les services numériques et le développement web[cite: 3, 8, 9].

## 2. Objectifs du Projet
* [cite_start]Héberger l'application web commerciale de vente de packs[cite: 14, 32].
* [cite_start]Déployer une application CRUD dédiée au département Ressources Humaines[cite: 15, 45].
* [cite_start]Sécuriser les services et les machines (Hardening)[cite: 16].
* [cite_start]Simuler l'infrastructure réseau via GNS3 ou Packet Tracer[cite: 17].
* [cite_start]Intégrer des pratiques DevOps (CI, versioning, GitHub)[cite: 18, 19].
* [cite_start]Identifier et corriger les faiblesses de sécurité via des tests d'intrusion[cite: 20, 131].

## 3. Architecture Technique
* [cite_start]**Système d'exploitation :** Linux (Ubuntu Server)[cite: 39, 52].
* [cite_start]**Serveur Web :** Nginx[cite: 40, 53].
* [cite_start]**Langages et Frameworks :** PHP (Laravel / Bagisto)[cite: 41, 54].
* [cite_start]**Base de données :** MySQL[cite: 42, 55].

## 4. Mesures de Sécurité
* [cite_start]Alignement sur les bonnes pratiques de la norme **ISO 27001**[cite: 11, 140].
* [cite_start]Chiffrement des données en transit via **HTTPS / TLS**[cite: 44, 73, 77].
* [cite_start]Segmentation réseau par **VLAN** et mise en place d'une **DMZ**[cite: 123, 124].
* [cite_start]Application du principe du moindre privilège et du **Zero Trust**[cite: 129].

## 5. Documentation technique
[cite_start]La documentation complète du projet, incluant les schémas d'infrastructure et les rapports de vulnérabilités, est réalisée avec **Docusaurus**[cite: 84, 140].
