# 🛍️ Ytech Secure E-Commerce Platform

> Application e-commerce moderne développée et personnalisée pour **Ytech Solutions** dans le cadre d’un projet de cybersécurité.

---

## 📌 Objectif

Cette plateforme permet de :

- 💼 Présenter les services digitaux de Ytech Solutions  
- 🛒 Vendre des packs de développement web  
- 📦 Gérer les commandes clients  
- 📩 Faciliter la communication avec les clients  

---

## ⚙️ Stack Technique

| Technologie | Description |
|------------|------------|
| 🐘 PHP (Laravel) | Framework backend |
| 🗄️ MySQL / MariaDB | Base de données |
| 🎨 Blade / Tailwind / Bootstrap | Interface utilisateur |
| 🌐 Apache / Nginx | Serveur web |
| 🔐 HTTPS (TLS) | Communication sécurisée |

---

## 🚀 Fonctionnalités principales

### 🛍️ Catalogue Produits
- Affichage des packs (Basic, Standard, Premium)
- Détails des services
- Navigation simple et rapide

### 🛒 Panier & Commandes
- Ajout / suppression de produits
- Calcul automatique du total
- Validation des commandes

### 👤 Gestion des utilisateurs
- Inscription & connexion sécurisée
- Gestion du profil
- Historique des commandes

### ⚙️ Administration (Back-office)
- Gestion des produits
- Gestion des catégories
- Gestion des clients
- Tableau de bord

---

## 🔐 Sécurité intégrée

L’application intègre plusieurs mécanismes de sécurité :

- 🔒 Authentification sécurisée
- 🛡️ Protection CSRF
- 🧼 Validation des données (inputs)
- 🔐 Hashage des mots de passe
- 🌐 Communication sécurisée via HTTPS
- 🚫 Protection contre XSS et injections SQL (via ORM Laravel)
